# aqi-predictor
AQI (Air Quality Index) Predictor using Machine Learning | Environmental Domain | Python, Pandas , Scikit-learn, Streamlit

AQI Predictor is a machine learning-based project designed to predict the Air Quality Index (AQI) for a given location using historical environmental data. The system analyzes key air pollutants — including PM2.5, PM10, NO₂, SO₂, CO, and O₃ — to forecast AQI levels, helping users understand and anticipate air quality conditions.
The goal of this project is to apply data science and machine learning techniques to a real-world environmental problem, making air quality insights more accessible and actionable. By training predictive models on cleaned and preprocessed pollution datasets, the system aims to provide reliable AQI estimates that can support awareness, planning, and informed decision-making around air quality.
This project was developed as a minor academic project, with a focus on end-to-end ML pipeline development — from data collection and preprocessing to model training and result visualization.
